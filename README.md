# 🌱 Agro Forte, Futuro Sustentável

Projeto desenvolvido para a competição Agrinho 2026 com o tema:

> **“Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”**

---

## 🎯 Objetivo do Projeto

Este site tem como objetivo **educar e conscientizar** sobre a importância do agronegócio sustentável, mostrando como a tecnologia, a inovação e as boas práticas agrícolas podem aumentar a produção de alimentos sem prejudicar o meio ambiente.

O projeto busca demonstrar que:

- É possível produzir em grande escala com responsabilidade ambiental
- A tecnologia é essencial para o futuro do campo
- Sustentabilidade e produtividade devem caminhar juntas

---

## 🌎 Temas abordados

- Agricultura sustentável
- Uso inteligente da água
- Energia renovável no campo
- Tecnologia agrícola (IA, drones, sensores e GPS)
- Conservação do solo e biodiversidade
- Evolução do agronegócio ao longo do tempo

---

## 🧠 Recursos do site

O projeto foi desenvolvido como uma **plataforma educativa interativa**, contendo:

### ✨ Interface moderna
- Design responsivo
- Efeitos de glassmorphism
- Gradientes animados
- Layout organizado por seções

### 📊 Conteúdo educativo
- Explicações sobre o agronegócio
- Cards informativos sobre tecnologia e sustentabilidade
- Linha do tempo da evolução do agro
- Curiosidades sobre o setor

### 🎮 Interatividade leve
- Scroll reveal (animações ao rolar a página)
- Contadores animados
- Quiz educativo sobre sustentabilidade
- Menu dinâmico com destaque da seção atual
- Botão de voltar ao topo

### 🎨 Experiência visual
- Fundo animado em gradiente
- Efeitos 3D em cards
- Partículas flutuantes
- Animações suaves de entrada
- Tipografia moderna (Poppins)

---

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando tecnologias web básicas:

- **HTML5** → Estrutura do site
- **CSS3** → Estilização, animações e responsividade
- **JavaScript (ES6)** → Interatividade e lógica
- **Google Fonts** → Tipografia moderna
- **Font Awesome** → Ícones
